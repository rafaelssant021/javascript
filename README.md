# Estudos em JavaScript

## Sobre
Este repositório reúne meus estudos, exercícios e práticas em JavaScript, desenvolvidos durante cursos e aprendizado contínuo.
O objetivo é consolidar conhecimentos fundamentais da linguagem e evoluir na lógica de programação e manipulação do DOM.

---

## Tecnologias utilizadas
- JavaScript
- HTML5
- CSS3

---

## Conteúdo
O repositório está organizado em exercícios e exemplos práticos, abordando:

- Variáveis e tipos de dados
- Estruturas condicionais
- Laços de repetição
- Funções
- Manipulação do DOM
- Eventos
- Lógica de programação

---

## O que estou aprendendo
- Fundamentos do JavaScript
- Interatividade em páginas web
- Resolução de problemas com lógica
- Organização de código

## Objetivo
Desenvolver uma base sólida em JavaScript para aplicação em projetos reais e evolução como desenvolvedor front-end.
